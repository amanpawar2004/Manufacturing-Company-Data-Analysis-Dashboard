# 🚀 Manufacturing Company Data Analysis Dashboard

## 📌 Project Overview

The **Manufacturing Company Data Analysis Dashboard** is an end-to-end data analytics project developed to analyze manufacturing operations and machine performance. Using Python for data cleaning and exploratory analysis and Power BI for interactive visualization, this project provides valuable insights into production efficiency, machine failures, defect trends, and operational performance.

The dashboard enables stakeholders to monitor key manufacturing metrics, identify patterns in machine failures, and support data-driven decision-making for improving productivity and reducing downtime.

---

## 🎯 Objectives

* Analyze manufacturing machine performance.
* Identify production efficiency and failure trends.
* Monitor machine operating conditions.
* Visualize important KPIs through an interactive dashboard.
* Generate insights for predictive maintenance and operational improvement.

## 📂 Dataset

**Dataset Name:** AI4I 2020 Predictive Maintenance Dataset

The dataset contains operational data collected from manufacturing machines, including:

* Air Temperature
* Process Temperature
* Rotational Speed (RPM)
* Torque
* Tool Wear
* Machine Failure Status
* Failure Types
